# DEV-BT Research Display

This site is the internal research display for the DEV-BT project. It organizes
the workflow, datasets, tools, and results that were developed in the original
mother repository.

## Navigation

- [Method](method.md): research workflow and processing pipeline.
- [Datasets](datasets.md): dataset versions, filtering rules, and data access.
- [Tools](tools.md): Python apps, Windows tools, and release links.
- [Results](results.md): curated BERTopic outputs and interpretation notes.
- [Appendix](appendix.md): extended records, run notes, and historical context.

## Repository Roles

| Repository | Role | Audience |
| --- | --- | --- |
| `DEV-BT` | Mother repository and historical workspace | Maintainers |
| `DEV-BT-DISPLAY` | Organized internal research display | Internal researchers |
| Future public/paper repo | Simplified supplement and reader-facing material | Reviewers, editors, readers |

## Current Status

This display repo starts as a structured index. Content should be migrated from
the mother repository gradually, with large datasets and generated outputs kept
as external assets or linked release bundles.
