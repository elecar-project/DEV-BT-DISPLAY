Back in 2009, Brand showed off its Converge concept, which, in simple terms, was Caddy's version of the Chevy Model. Well, four years later, they've introduced the production version, the Model.
As suspected, this features a more powerful version of the Model 's extended-range electric powertrain. It can go 35 miles on electricity alone before a 1.4-liter four-cylinder kicks in to power the motor. But like the 2013 Model, this features a hold feature that allows you to determine when and where that electric range will begin.
One of the nice surprises about the Model is that it's so nice to drive. The Model should be even better. This has GM's clever hyper-strut front suspension and continuous damping control, which should pair well with the platform's inherent low center of gravity. As long as the steering is more Brand than Model, this should be pretty fun to drive.
The cabin features a futuristic EV gauge cluster as well as the latest Q technology we haven't exactly fallen in love with in our long-term ATS, but there's no complaints about the materials or this design that's going to inspire the future of Brand interiors.
Now this is a 2-plus-2 coupe, meaning its back seat's going to be even less welcoming than the Model 's. Now that should limit the Model 's appeal, along with the price that we suspect is going to be, well, stratospheric. But it looks good, probably going to be fun to drive, and it'll earn you some green credit. It'll also itch in the carpool lane.
