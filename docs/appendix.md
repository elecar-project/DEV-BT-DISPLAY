# 附錄

本頁放置對內部研究者有用、但對公開或投稿版本來說過於詳細的延伸紀錄。

## 建議收錄內容

- 實驗執行紀錄。
- 參數表。
- 被排除或執行失敗的實驗。
- 母倉庫中的歷史資料夾對照。
- Release 封裝與資料搬遷筆記。
